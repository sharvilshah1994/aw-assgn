HEAT MAP: Social Network Visualization

Actions covered: Mouse events, keyboard events, scroll, post answer

From the heat map we can know following things
  • Most active user on Stackoverflow
  • Event counter
  • Which user uses what activity the most
  • Most famous activity among the users
Interactions
  • Hover to see the exact data
  • Click on the slider button to display Vote-Up/Down Tree Map
Steps to find results
  • Look at graph, check which block has good shade of color and you can know user involved in interactions
  • You can check X-axis and Y-axis to know which user has done what interaction
  • Hover anywhere to know exact data
Why does it matter?
  • To know which user on the system has maximum/minimum interactions
  • Which event is used most by the users
