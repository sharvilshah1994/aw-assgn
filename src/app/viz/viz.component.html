PIE CHART: Graph for the Logged in user

Actions covered: Mouse events, keyboard events, scroll, post answer

From the pie chart we can know following things
  • Number of events
  • What activity has user done most till now
  • Total number of user events done till now
Interactions
  • Hover to see the exact data
  • Click on the graph to see all-time interactions
  • Click on the radio button to change graphs
  • Click on the slider button to display social network visualization
Steps to find results
  • Hover over the pie chart to get exact counter
  • Look for the event that covers maximum portion and look at legend to know which event is that
  • You can also look at percentage to know answers
Why does it matter?
  • We can know which event is most important for the user and optimize the application in that way
  • We can also know from here if there is any anomaly

BAR CHART: Graph for Logged in user

Actions covered: Mouse events, keyboard events, scroll, post answer

From the bar chart we can know the following things
  • Date of the particular event
  • On what date was the user interaction highest
  • Which event has the highest counter
Interactions
  • Hover to see the exact data
  • Click on the graph to see date-wise interaction
  • Click on the radio button to change graphs
  • Click on the slider button to display social network visualization
Steps to find results
  • By looking at graph, you can see which bar is bigger
  • You can get date from X-axis and count from Y-axis
  • You can also hover graph and get results
Why does it matter?
  • It matters because it is important to know the date when user was active
  • Also, which event user uses the most to track activities. We can immediately detect anomaly

SPIDER CHART: Social Network Visualization

Actions covered: Mouse events, keyboard events, scroll, post answer

From the bar chart we can know the following things
  • Comparison of events between different users
  • Which event has highest priority among all users
  • Which event has the highest counter
  • Most active user on Stackoverflow
Interactions
  • Hover to see the exact data
  • Click on the legend to eliminate any user
  • Click on radio button to show current user's all-time or date-wise logs
Steps to find results
  • Look at graph, check which portion covers maximum area and you can know the user who has maximum interactions
  • Click anywhere on the legend and you can eliminate any user interaction on the graph to know exact answer
  • Hover anywhere to know exact data
Why does it matter?
  • To know which user on the system has maximum/minimum interactions
  • Which event is used most by the users
